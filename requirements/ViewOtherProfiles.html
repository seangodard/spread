Spread: Use Case #19

Title

View Other Profiles

Actors

User

Prerequisite

None

Action

The user selects the account holder's profile they wants to view, whether that is through clicking on another account holder's name on a video they are viewing, or through searching for an account holder using the search function.

Effect

User is taken to the account holder's profile page.

Next

No required follow-up actions.
