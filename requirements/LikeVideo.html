Spread: Use Case #18

Title

Like Video

Actors

User

Prerequisite

The user is on a page where an embedded video is present.

Action

The user clicks the Like Video button on the video they want to like.

Effect

The Like Count that is displayed by the video's like button increments by one.

Next

No required follow-up actions.
