Spread: Use Case #2

Title

Log In

Actors

User

Prerequisite

The user must have already created an account.

Action

The user will input their username and password.

Effect

If the username and password are correct, they will be logged in to their account. If not, they will be informed that an error has occurred. They are redirected to the home page.
