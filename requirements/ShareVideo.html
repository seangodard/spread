Spread: Use Case #20

Title

Share Video

Actors

User

Prerequisite

The user must have a video on their current page.

Action

User clicks on one of the share buttons for their desired social media site.

Effect

The user is taken to the selected social media site.

Next

The user interacts with the social media site's sharing/posting function.
